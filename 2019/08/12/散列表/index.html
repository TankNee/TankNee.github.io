散列表相关的几个算法

根据算法图解一书写的两个算法：广度优先算法以及狄克斯特拉算法：

散列表

散列表在Python中也称作字典，通常的定义方式为： name = dict()

也可以写作： name = {}

散列表中的每一项都由Key(键)和Value(值)组成。散列表的搜寻效率在一般情况下接近O(1)而在最糟糕的情况下为O(n),可见是一个非常好的数据结构。

广度优先算法

广度优先算法顾名思义就是优先寻找临近的可能对象，当临近的对象均不是所搜寻的目标时，再去寻找较远的可能对象。
